O que é HTML? ↑Voltar

HTML significa HyperText Markup Language e foi criado pelo britânico Tim Berners-Lee. O HTML é o componente básico de uma página da web, ele permite inserir o conteúdo e estruturar a página de um website Sem o HTML para organizar e dar significado às informações de uma página, o navegador não saberia exibir textos, carregar imagens, vídeos e outros conteúdos.

Os hipertextos são conjuntos de elementos conectados. Podendo ser palavras, vídeos, imagens, documentos, áudio e etc. Quando conectamos todo esse conteúdo, formamos uma rede de informações que permite organizar conhecimentos e guardar informações.

Quando visitamos um website, percebemos diferentes distribuições e tamanhos para títulos, parágrafos, imagens, vídeos e outros elementos. A estruturação é feita através do HTML. No início da web era comum encontrar sites apenas com textos e imagens simples, com estruturação básica e sem estilizações complexas, mas hoje em dia quase não encontramos sites que possuam apenas elementos HTML e, por tanto, podemos considerar que o HTML é apenas o esqueleto da página. Além do esqueleto é necessário ter o corpo e para isso temos o CSS e o JavaScript que, em conjunto com o HTML, dão a vida aos sites como conhecemos hoje.

Portanto podemos concluir que o HTML é um arquivo de extensão .html ou .htm com a estrutura do site que o navegador irá interpretar para renderizar o conteúdo e o usuário final consiga visualizá-lo.

Estrutura básica ↑Voltar

O documento HTML é composto por tags. Começando com a tag <html> e terminando com </html>. Dentro da <html> temos a tag <head></head> que é onde fica o cabeçalho e o título que aparece na janela do navegador e a tag <body></body> que é onde colocamos o conteúdo que será vísivel no corpo da nossa página como pode ser visto no exemplo abaixo:

O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Sempre será dessa forma e você sempre, sempre começará seu HTML começando por esse código. Geralmente os editores como o Sublime Text já tem atalhos para iniciar os documentos HTMLs com essa estrutura, logo, você não precisa se preocupar em decorá-la, mas é bom que faça.

Tags ↑Voltar

As tags são usadas para informar ao navegador a estrutura do site, ou seja, quando o navegador interpreta o código HTML ele produz a estrutura e o conteúdo visual da página. Elas sempre estão dentro dos sinais de chevron < > (maior que e menor que).

Algumas tags precisam de fechamento e outras não. As que precisam de fechamento são escritas como <tag></tag> e as que não precisam de fechamento são escritas como <tag />.

Além disso as tag podem receber atributos que possuirão valores que modificam sua estrutura ou funcionalidade

Existem tags para todo tipo de objeto, extruturação da página e conteúdo. Logo abaixo veremos alguns exemplos de tags de Textos, de Listas e de Links.

Textos ↑Voltar

Vamos iniciar com a tag de Parágrafo <p>. Essa linha que você está lendo está dentro dessa tag, criando um parágrafo separado e iniciando uma nova linha

Temos também as tags de título e subtitulo que possuem 6 variações, são elas: <h1><h2> <h3><h4><h5><h6>

Além dessas tags, temos as tags de formatação de texto, temos a tag <sub> que deixa o texto abaixo da linha padrão, temos a <sup> que deixa acima, temos a tag <strong> que deixa o texto negrito, temos a tag <i> que deixa o texto em itálico, a tag <u> faz com que o texto fique sublinhado, a tag <del> que faz com que o texto fique tracejado e, por fim, temos a tag <mark> que destaca nosso texto como se usasse um marca-textos.

Também temos a tag <blockquote> que cria uma citação com um espaçamento no começo da linha.

Links ↑Voltar

Por fim, temos a tag de hyperlink, que faz com que você consiga navegar de um ponto a outro ou de uma página para outra. Nesse caso utilizamos a tag <a> e dentro dela passamos o atributo href para direcionar o usuário ao clicar. Como exemplo, se tivermos

<a href="https://google.com/">Google</a>

teremos como resultado: Google.

Você também pode usar no atributo href um id de um elemento da página, como fizemos com o botão "Voltar" acima, direcionando para o índice da página e não para um site externo.

Dentro dessa tag temos alguns atributos interessantes, como por exemplo o target que você pode fazer o link abrir em uma nova guia e o title que exibe um texto quando você para o mouse sobre ele (chamado de tooltip). Vamos ver como ficaria nosso exemplo anterior adicionando esses dois elementos.

<a href="https://google.com/" title="Clique para ir ao Google" target="_blank">Google</a>

teremos como resultado: Google
